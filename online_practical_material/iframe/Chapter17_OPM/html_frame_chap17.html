"ABC_PCM.c": A program code used in an analysis of weevil rostrum length in Chapter 17. Programs were written in C and should be run in a Unix (Linux) machine. All data can be input in the .c file. Each branch is labeled with consecutive numbers, which corresponded with its descendent node (shown in a parenthesis on the phylogeny below).

Codes

Since we are interested in estimating trait values of internal nodes at the end of branch 0 and 1, we output a value of MRCA (theta), P[0], and P[1] (corresponding to species A, B, and C in Fig 17.2).

Compile the C code as

>> gcc ABC_PCM.c

Run the program

>> ./a.out
